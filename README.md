# yang
 idea 集思录东方财富可转债列表,待发转债插件
 
 ## 使用方法
 在https://github.com/xbb2yy/yang/releases 下载最新的压缩包,拖到自己的idea 中即可
 在idea 中双击 shift,输入yang 就可以找到本插件
 
 * 可转债列表
 > 如果需要获取完整数据,需要设置集思录用户名密码
